# Description
The progect draws an ellipse with given semi-major and semi-minor axies.

The semi-major axis:a=7

The semi-minor axis:b=3

The y-coordinate(the vertical coordinate) is presented as function of the x-coordinate(the horizontal coordinate) using the 
ellipse's canonical equation which gives connection between the y-coordinate and the x-coordinate using the semi-major and 
semi-minor axies.
# Features
math-based program

Graph visualisation using matplotlib.pyplot

NumPy calculations
